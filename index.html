Demo av funktionen calc()

Funktionen calc() i CSS kan användas för att beräkna saker, t.ex. storlek på element.

Den här vita huvuddelen av sidan har minsta höjden (min-height) satt till calc(100vh - 80px - 20vw). 

          
main {
  min-height: calc(100vh - 80px - 20vw);
}

Det gör att den kommer fylla upp precis så mycket höjd som "finns kvar" i viewport efter att headern använt sina 80px höjd och footern använt sina 20vw höjd. calc() kan alltså räkna med olika enheter!

calc() kan räkna alla fyra räknesätten: plus(+), minus(-), gånger(*) och delat(/).
